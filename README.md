# 🚑 Rapid Crisis Response System (RCRS)

## 📌 Overview
Rapid Crisis Response System (RCRS) is a web-based platform designed to provide quick assistance during emergency situations. It helps users access nearby hospitals, navigate efficiently, and take immediate action using a simple interface.

## ❗ Problem Statement
In emergency situations, delays in response and lack of quick access to nearby medical facilities can lead to serious consequences. There is a need for a system that enables fast decision-making and easy access to emergency services.

## 💡 Solution
RCRS provides a unified platform where users can:
- Trigger emergency (SOS) quickly  
- Find nearby hospitals with distance and ETA  
- Navigate using the fastest route  
- Contact hospitals directly  

This reduces response time and improves accessibility.

## ✨ Features
- 🔴 SOS emergency activation  
- 🏥 Nearby hospital detection  
- 📞 Direct call option  
- 🧭 Route navigation  
- 🎤 Voice input support  
- 📝 Text-based symptom input  
- 📊 Dashboard for monitoring  

## ⚙️ Tech Stack
- HTML5  
- CSS3  
- JavaScript
